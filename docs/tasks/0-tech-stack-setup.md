## Decisions

### Local Postgres port

**Context:** Port 5432 was already in use by another container on this machine.
**Decision:** Use port 5434 for this project's local Postgres.
**Alternatives considered:** Could stop the other container, but it likely belongs to another project.

### Postgres driver

**Context:** Need Neon serverless driver on Vercel and node-postgres locally.
**Decision:** Dual-driver setup in `lib/db/index.ts` — checks `process.env.VERCEL` to select `drizzle-orm/neon-serverless` (prod) vs `drizzle-orm/node-postgres` (local).
**Alternatives considered:** Single driver with postgres.js — wouldn't work optimally on Vercel's edge/serverless.

### dotenv for Drizzle Kit

**Context:** Drizzle Kit runs outside of Next.js and doesn't load `.env.local` automatically.
**Decision:** Added `dotenv` as a dev dependency and load `.env.local` in `drizzle.config.ts`.
**Alternatives considered:** Inline env vars in scripts — less ergonomic.
